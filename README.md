# installation

bower i jquery.tabs

# usage

requirejs supported

you can group them any way

``` html
<div class="tabs">
	<div class="tabs__toggle tabs__toggle_active">tab1</div>
	<div class="tabs__toggle">tab2</div>
	<div class="tabs__tab"></div>
	<div class="tabs__tab"></div>
</div>

```
